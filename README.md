# Gerenciamento-de-Estoque-C-Sharp-
Desenvolvi um sistema desktop de controle de estoque em C# com Windows Forms, com cadastro, edição, busca e status de produtos. Usa SQLite para persistência local, mantendo dados após fechar o app. Possui filtros, histórico de entregues e backup em JSON. Projeto focado em POO, CRUD e integração com banco de dados.
